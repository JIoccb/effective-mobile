```python
def is_isomorphic(s: str, t: str) -> bool:
    return len(s) == len(t) and len(set(s)) == len(set(t)) == len(set(zip(s, t)))
```
### Оценка по времени 
Один проход по строкам — $O(n)$, где $n$ — длина строки. Операции с множеством в среднем $O(1)$. Это оптимально: чтобы вынести вердикт, нужно прочитать каждый символ хотя бы раз, поэтому $\Omega(n)$ — нижняя граница.

### Оценка по памяти
Хранятся два множества размером с число различных символов $k$, то есть $O(k)$, причём $k \le \min(n, |\Sigma|)$, где $|\Sigma|$ — размер алфавита.
